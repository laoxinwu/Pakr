# Features

| Feature | Description |
|---|---|
| 🔨 Fully automated build | Trigger GitHub Actions to compile, sign, and package automatically |
| 📊 Real-time progress | Display the exact percentage and current build stage |
| 📱 Full-screen WebView | Provide an immersive experience with iOS-style loading animation and adaptive progress color |
| 🔏 Release signing | Automatically sign with a keystore to support upgrade installs |
| 📦 Multi-ABI output | Generate both arm64-v8a and armeabi-v7a APKs |
| 🌐 Any website support | Support HTTP / HTTPS, cookies, file upload, and camera permission |
| ⬇️ System downloads | Forward website downloads to Android DownloadManager |
| 🔄 Pull to refresh | Refresh the page when pulled down at the top |
| ⌨️ Keyboard adaptation | Move the page up automatically when the soft keyboard opens |
| 🗂️ Build history | Save recent packaging history and support one-click refill |
| 🌙 Dark mode | Follow the system theme and support manual switching |
